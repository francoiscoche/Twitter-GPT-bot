# Script allows you to generate and post a tweet

- Tweet generated via the ChatGPT API.
- Tweets posted via the twitter API



&nbsp;


## Run localy the project



Install dependencies
```bash
npm install
```

copy rename and complete .env.example (.env)
```bash
CONSUMER_KEY=
CONSUMER_SECRET=
ACCESS_TOKEN_KEY=
ACCESS_TOKEN_SECRET=
OPENAI_API_KEY=
```

Run the script
```bash
node index.js
```
